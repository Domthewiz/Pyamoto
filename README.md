# Pyamoto
## The New Super Mario Bros. U level editor
Pyamoto is an advanced fork of the original Miyamoto editor with the purpose of improving functionality and usability. 

## Installation
You can download the latest release: https://github.com/Zenith-Team/Pyamoto/releases/latest

## Usage
Follow the guide on the wiki: https://zenith.nsmbu.net/wiki/Miyamoto_Level_Editor

## Running from Source
1. Install the latest version of [Python 3](https://www.python.org/downloads/) (on Windows, select the option to add Python to PATH)
2. Clone the repo: `git clone https://github.com/Zenith-Team/Pyamoto`
3. Create a .venv: `cd Pyamoto && python -m venv .venv`
4. Activate: `./.venv/scripts/activate.bat` (windows) or `source ./.venv/bin/activate` (macOS/linux) You should now see (.venv) at the start of your terminal
5. Install deps: `pip install -r requirements.txt`
6. Run: `python pyamoto.py`
